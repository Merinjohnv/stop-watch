## Stopwatch Web App

A simple and responsive Stopwatch application built using HTML, CSS, and JavaScript.
It allows users to start, stop, reset, and record lap times with smooth real-time updates.

---

## Features

Start / Stop / Reset controls

Lap time recording

Real-time time updates using setInterval()

Clean and responsive UI

Beginner-friendly project

---

## Tech Stack

HTML

CSS

JavaScript

Working with performance.now() for accurate timing

DOM manipulation
